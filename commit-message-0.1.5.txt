feat: Version 0.1.5 - Support Linux Mint et amélioration détection OS

## Corrections

### Détection OS
- **Détection Linux Mint** : Correction de la détection OS pour les systèmes Linux Mint
  - Amélioration de la logique de détection pour vérifier Mint avant Ubuntu (Mint a `ID_LIKE="ubuntu debian"`)
  - Ajout d'une fonction helper `idLikeContains()` pour gérer correctement les champs `ID_LIKE` multi-valeurs
  - Détecte maintenant correctement Mint même si seul `ID_LIKE` contient "ubuntu"
  - Supporte la détection via `ID=linuxmint`, `ID=mint`, ou `ID_LIKE` contenant "mint"/"linuxmint"

## Ajouts

### Support Linux Mint
- **Icône Mint** : Ajout de l'icône Linux Mint dans les icônes de plugins frontend
  - L'icône Mint s'affiche correctement dans la section de gestion des plugins
  - L'icône s'affiche automatiquement lorsque l'OS est détecté comme Mint
  - Utilise le fichier d'icône existant `mint.svg`

- **Fichiers de logs spécifiques Mint** : Ajout de `mintupdate.log` aux fichiers de logs système par défaut pour Mint
  - `/var/log/mintupdate.log` maintenant inclus dans la liste des fichiers de logs par défaut pour Mint
  - Ajouté aux patterns de fichiers par défaut (`mintupdate.log*`) pour la détection automatique
  - Inclus dans la détection de rotation des logs pour les systèmes Mint
  - Le fichier est désactivé par défaut mais visible dans la liste des fichiers de logs

## Modifications

### Logique de détection OS
- **Ordre de détection** : Réorganisation de la détection OS pour vérifier Mint avant Ubuntu
  - Empêche Mint d'être incorrectement détecté comme Ubuntu
  - Identification plus précise du type d'OS pour les systèmes basés sur Mint
  - Meilleure gestion du champ `ID_LIKE` avec plusieurs valeurs
